# ⚔️ SPC PURGE
This purge system is very simular to the purge system from Elitelupus Roleplay.<br>
I took inspiration from the looks and turned it into a completely free, public addon for anyone to use in servers.<br>
Down below is the requirements and what console commands there are.

# 📜 INFORMATION
You will need [DarkRP](https://github.com/FPtje/DarkRP) and [PIXEL-UI](https://github.com/TomDotBat/PIXEL-UI) to run this addon.

| Console Commands       | State    | Permission | Description |
| :-------- | :------- | :-------- | :-------------------------------- |
| `spc_purge_forceidle` | `IDLE` | `superadmin` | Sets the purge state to idle. |
| `spc_purge_forceprepare` | `PREPARE` | `superadmin` | Sets the purge state to prepare. |
| `spc_purge_forceactive` | `ACTIVE` | `superadmin` | Sets the purge state to active. |
| `spc_purge_forcegrace` | `GRACE` | `superadmin` | Sets the purge state to grace. |
